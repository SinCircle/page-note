# PageNote - 页面便签

一个轻量的 Chrome / Edge 浏览器扩展，用来在任意网页上快速放置可编辑、可拖拽、可调整大小的手写风格便签。

## 功能特点

- 一键在当前页面插入便签
- 便签支持直接编辑纯文本内容
- 支持拖拽移动位置
- 支持右下角手柄调整尺寸（左右两侧对称延伸）
- 内置手写字体和横线纸张风格
- 删除便签时带有淡出动画
- 通过页面内注入样式提高网页截图兼容性
- 便签内容自动持久化，刷新页面后恢复
- 支持固定模式（蓝点）：便签固定在屏幕视口，不随页面滚动
- 支持单行模式：高度压缩至最小时自动切换为无横线居中单行样式，并随内容横向自适应宽度
- 多行模式下，在上方空白处左侧悬停显示统计信息（格式：`词数|字数`）

## 当前限制

- 浏览器内部页面（如 `chrome://`、扩展页、商店页）通常无法注入内容脚本

## 安装方法

1. 下载或克隆本项目到本地
2. 打开浏览器并进入 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择当前目录中的 `page-note` 文件夹

## 使用方法

1. 打开任意普通网页
2. 点击浏览器工具栏中的 PageNote 图标
3. 页面中会生成一个新的便签
4. 直接点击便签内容区域即可输入文字
5. 悬停便签后，可使用顶部拖拽区域移动位置
6. 可使用右下角手柄调整便签大小
7. 点击左上黄点可切换固定 / 跟随页面模式
8. 多行模式下悬停便签时，可在左上方空白处看到 `词数|字数` 统计
9. 点击右上角红色按钮删除便签

## 项目结构

- `manifest.json`：扩展配置，定义权限、图标、后台脚本和内容脚本
- `background.js`：监听工具栏点击事件并通知当前标签页创建便签
- `content.js`：负责创建便签、编辑、拖拽、缩放和删除逻辑
- `content.css`：便签样式与动画
- `icons/`：扩展图标资源

## 技术栈

- Manifest V3
- Vanilla JavaScript
- Chrome Extension APIs
- CSS 动画与内容脚本注入

## 适用场景

- 临时记录网页信息
- 给阅读中的页面添加标注
- 做截图前的页面批注或提示
