# PageNote - 页面便签

一个轻量的 Chrome / Edge 浏览器扩展，用来在任意网页上快速放置可编辑、可拖拽、可调整大小的手写风格便签。

## 功能特点

- 一键在当前页面插入便签
- 便签内容支持富文本编辑（contenteditable），粘贴时自动转为纯文本
- 支持拖拽移动位置
- 右下角手柄调整大小：向下拖动改变高度，向右拖动以中心对称拉伸宽度
- 单行模式：高度压缩至极小时自动切换为无横线、文字水平居中的单行标签样式
- 固定 / 跟随切换：小黄点按钮可在"固定在屏幕（position: fixed）"和"跟随页面滚动（position: absolute）"之间切换；固定后小黄点变蓝
- 便签的位置、尺寸和内容自动持久化到 `chrome.storage.local`，刷新或重开标签页后自动恢复
- 内置 Caveat / Long Cang 手写字体与横线纸张风格
- 删除便签时带有弹出与淡出动画
- 关键样式同时注入页面 DOM，提升 Edge 自带网页截图对便签的兼容性

## 当前限制

- 浏览器内部页面（如 `chrome://`、扩展页、商店页）无法注入内容脚本，便签功能不可用
- 便签数据以页面 URL（去掉 hash）为 key 存储，URL 变化后原便签不会自动迁移

## 安装方法

1. 下载或克隆本项目到本地
2. 打开浏览器并进入 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择当前目录中的 `page-note` 文件夹

## 使用方法

1. 打开任意普通网页
2. 点击浏览器工具栏中的 PageNote 图标，页面中会生成一个新的便签
3. 直接点击便签内容区域即可输入文字
4. 悬停便签后：
   - 顶部灰色把手：拖拽移动便签位置
   - 右上角**小黄点**：点击切换固定 / 跟随页面模式（固定后变蓝）
   - 右上角**小红点**：删除便签
   - 右下角**斜纹手柄**：拖拽调整便签大小
5. 将便签高度拖至最小（约 36 px）可进入单行标签模式；向下拖动高度恢复多行模式
6. 所有便签状态在刷新或重开标签页后自动恢复

## 项目结构

- `manifest.json`：扩展配置，定义权限（`activeTab`、`storage`）、图标、后台脚本和内容脚本
- `background.js`：监听工具栏点击事件，过滤非 http(s) 页面后向当前标签页发送创建便签消息
- `content.js`：便签创建、编辑、拖拽、缩放、固定切换、单行模式、持久化存储与恢复逻辑
- `content.css`：便签样式（含单行模式、固定模式、拖拽 / resize 中禁止选中等）与动画
- `icons/`：扩展图标资源

## 技术栈

- Manifest V3
- Vanilla JavaScript
- Chrome Extension APIs（`chrome.action`、`chrome.tabs`、`chrome.storage.local`、`chrome.runtime`）
- CSS 动画与内容脚本注入

## 适用场景

- 临时记录网页信息，关闭后仍可恢复
- 给阅读中的页面添加标注
- 做截图前的页面批注或提示
- 固定常驻提示卡片在屏幕固定位置
